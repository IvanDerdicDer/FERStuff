ADMIN
-----------------------------------------
Add user:
- python3 usermgmt.py -a/--add -u/--username <username>

Change user password:
- python3 usermgmt.py -pass/--passwd -u/--username <username>

Force password change:
- python3 usermgmt.py -fpass/--force_passwd -u/--username <username>

Delete user:
- python3 usermgmt -d/--delete -u/--username <username>


USER
-----------------------------------------
Login:
- python3 login.py -u <username>
